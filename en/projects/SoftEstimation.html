Hardness estimation from depressing behaviour movie using deep learning

In this study, we propose a hardness estimation method from the movements of a human hand and arm that depresses the object with a chopstick using deep learning, which is useful for the scene where the user wants to know the hardness of the object without touching it (cooking, etc.).

The evaluation experiments show the hardness value of an unknown hardness object can be estimated accurately by using a model that has been learned the the pressing motion video for each user for multiple hardness objects.

This method is new as it is the first to realize indirect estimation that does not use the information of the object itself, because it estimates only from the part of the hand and arm without the object.
